pip install pytest pytest-django 

create a file : pytest.ini

content in the file: 
    [pytest]
    DJANGO_SETTINGS_MODULE = project.settings
    python_files = test.py tests.py test_*.py tests_*.py *_test.py *_tests.py
    addopts = 
        --doctest-modules
        --strict-markers
    markes = 
        slow: Run tests that are slow
        fast: Run fast tests
